# 뷰포트 조작
🌟 (W or A or S or D) :

카메라 각도에 평행하게 좌우앞뒤로 이동

🌟 마우스 우클릭 + (Q or E) :

카메라 각도에 수직이게 위 아래로 이동

🌟 액터클릭 + F :
액터 포커싱

# 액터 배치 단축키
🌟 액터 클릭 + end :

액터 내려 놓기

🌟 Alt + 액터 트랜스폼 조정 :
액터 복제

# 에디터 단축키
🌟 (W or E or R) :

트랜스폼 단축키

Alt + P :

플레이

Esc :

종료

Shift + F1 :

플레이 중 마우스 커서 보기

Ctrl + Space :

콘텐츠 브라우저 불러오기
